Unit Converter in C (Beginner-Friendly Project)

A simple and interactive Unit Converter program written in C, using menus and switch statements to convert between various length, weight, and temperature units.

This project is designed for beginners learning:

Functions

Switch-case decision making

User input validation

Modular program structure

Loops and menus
